pISA-tree: Standard project directory tree
(ISA-tab compliant)

This set of batch files is used to create standard directory tree for research projects. Batch files are provided to make standardized nested directories in accordance with the ISA-tab framework (Investigation/Study/Assay).

Installation:

  1. Download (zip file) or clone the repository from NIB GitHub repository.
  2. Extract files into your local pISA projects directory (ignore or delete .git-related files).
  3. User instructions are in INSTRUCTIONS file.
